**概述：**你是否用到过这些css呢...？

#### Selectors
Root
```scss
:root { }
```
使用root可以让你在DOM中选择最高级父元素。如果你在写HTML,那么html元素永远是最高的。

因为对于一个目标元素伪类选择器有更高特性，你可以用:root重写<html>元素。
